# 输入文件说明

将业务表格放入此目录，程序会自动识别所有 `.xlsx` 文件。

## 表格格式要求

| 提交时间 | 商品SKU | 提交者 | 价格 | 图片 | 备注 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 2026/6/8 20:00 | 100264886683 | 用户A | （空） | （空） | （空） |
| 2026/6/8 20:00 | 10069951490519 | 用户B | （空） | （空） | （空） |

## 注意事项

- 必须包含 `商品SKU` 列
- 价格、图片、备注列可为空，程序会自动填充
- 支持多个表格文件，运行时会列出供选择
