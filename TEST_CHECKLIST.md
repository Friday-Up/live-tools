# 全链路测试节点清单

## 一、首次使用链路（用户第一次拿到产品）

### 1. 下载安装阶段
- [ ] 从 GitHub Release 下载 ZIP 文件
- [ ] 解压 ZIP 文件
- [ ] 检查解压后的文件完整性
  - SKU-Price-Audit-Web.exe
  - 启动测价工具.bat
  - templates/index.html
  - config.py
  - utils/ 目录
  - input/ 目录

### 2. 启动阶段
- [ ] 双击"启动测价工具.bat"
- [ ] 检查终端窗口是否显示启动日志
- [ ] 检查是否自动打开浏览器访问 http://localhost:8080
- [ ] 检查页面是否正常加载（不出现 Internal Server Error）

### 3. 首次登录阶段
- [ ] 上传 Excel 文件
- [ ] 设置价格门槛
- [ ] 点击"开始测价"
- [ ] 检查是否弹出浏览器窗口
- [ ] 检查是否显示"需要登录"提示
- [ ] 在浏览器中完成京东登录
- [ ] 检查是否检测到已登录状态
- [ ] 检查是否继续执行测价任务

### 4. 测价执行阶段
- [ ] 检查进度条是否正常更新
- [ ] 检查日志是否正常输出
- [ ] 检查是否正确识别价格
- [ ] 检查是否正确标记低于门槛价的商品

### 5. 结果输出阶段
- [ ] 检查是否生成结果文件
- [ ] 检查是否可以下载结果
- [ ] 检查 Excel 文件内容是否正确

### 6. 停止测价阶段
- [ ] 点击"停止测价"按钮
- [ ] 检查浏览器是否关闭
- [ ] 检查程序是否停止运行

---

## 二、二次使用链路（已有登录态）

### 1. 启动阶段
- [ ] 双击"启动测价工具.bat"
- [ ] 检查是否自动加载上次登录态

### 2. 登录态检测阶段
- [ ] 检查是否自动检测到已登录状态
- [ ] 检查是否跳过登录步骤直接开始测价

---

## 三、登录态失效链路

### 1. 登录态失效检测
- [ ] 启动测价任务
- [ ] 检查是否检测到登录态失效
- [ ] 检查是否显示"我已登录"弹窗

### 2. 重新登录
- [ ] 在浏览器中重新登录
- [ ] 点击"我已登录，继续"按钮
- [ ] 检查是否继续执行测价任务

---

## 四、异常情况链路

### 1. 无输入文件
- [ ] 不上传文件直接点击开始
- [ ] 检查是否有错误提示

### 2. 格式错误文件
- [ ] 上传非 xlsx 文件
- [ ] 检查是否有错误提示

### 3. 网络异常
- [ ] 断网情况下启动测价
- [ ] 检查是否有错误提示

### 4. 浏览器异常
- [ ] 手动关闭浏览器窗口
- [ ] 检查程序是否能正常处理
